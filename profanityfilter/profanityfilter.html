profanityfilter

A universal Python library for detecting and filtering profane words.

GitHub: https://github.com/areebbeigh/profanityfilter
PyPI: https://pypi.python.org/pypi/profanityfilter

Installation

Installing is easy:
> pip install profanityfilter

or if you downloaded the source from GitHub
> python setup.py install
